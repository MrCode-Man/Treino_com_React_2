# React + Vite

Esse repositório é dedicado para meu caso de estudo com React + Vite.  

O site que estou estudando está [aqui](https://designcenter.calarts.edu/?ref=onepagelove)

Como pré-requisitos, você tem que ter instalado no seu computador o node `v18.16.1` ou mais recente, você consegue instalar ele por [aqui](https://nodejs.org/pt-br/download), para garantir que você instalou tudo da forma certa procure um vídeo de instalação dele.

---

## Como executar o programa?  

Você vai apertar numa caixa verde escrita: `Code` e `Download zip`  

Após isso, você vai extrair o seu .zip e acessar ele. Você tem duas opções:
* Abrir a pasta em um [editor de código](#Editor-de-Código)
* Abrir no [CMD](#CMD)

Após aberto, se você escolheu usar o editor de código, abra um terminal no seu editor, rode o comando: `cd (aperta tab)` e enter  

Para garantir que estamos na mesma pasta, rode um `dir` ou `ls` e veja se aparece a pasta `src` e outras, se sim você está no caminho certo  

Agora, rode:
```
npm install
npm run dev
```

O seu terminal vai mostrar uma linha como essa:

```
> desculpaif@0.0.0 dev
> vite


  VITE v5.4.21  ready in 474 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
  ➜  press h + enter to show help
```

Segure `Ctrl` e aperte nesse http mostrado no terminal com o mouse e você vai ser direcionado para o site criado a partir do código.

### Editor de Código
Aqui também tem duas opções:
* Abrir pelo [explorer](#Explorer)
* Abrir pelo [editor](#Editor)

### CMD
Esse é mais direto, você vai apertar nele com o botão direito e ir em `copiar como caminho`  

Dentro do seu cmd você vai digitar `cd (caminho copiado)` e aperte enter

---

#### Explorer
A pasta que você extraiu vai estar com o nome: `Treino_com_React--main`. Aperte com o botão direito do mouse nela.  

Caso esteja no Windows 10 vai aparecer a opção `open folder as (seu editor)`  

Se for no Windows 11, você vai em `mostrar mais opções` e aperte na opção `open folder as (seu editor)`

#### Editor
Abra o seu editor, procure por `Open folder`. Procure pela pasta extraída e toque duas vezes nela
